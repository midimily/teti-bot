# Teti.bot wordmark source

`teti-bot-wordmark.svg` contains the final outlined wordmark used by the desktop app. It was generated from the official Inter 4.1 variable font at weight 780 and optical size 20, with the `i` dot separated into the accent path.

Inter is Copyright 2016–2024 The Inter Project Authors and is licensed under the [SIL Open Font License 1.1](https://github.com/rsms/inter/blob/v4.1/LICENSE.txt). The OFL permits documents produced with the font; Teti stores only the resulting SVG outlines and does not bundle or redistribute the Inter font software.

The SVG has a tight `0 0 7585 1579` viewBox. Its body and accent paths intentionally use separate Teti CSS color tokens.
